Sorts a list of non-negative integers in place using Radix Sort.

This is an implementation of LSB-first Radix Sort, optimized for 32-bit integers. It is a stable sort.

data is the list of integers to be sorted. The elements are assumed to be non-negative and within the 32-bit unsigned integer range. If reuseBuffer is true, the auxiliary buffer will be sourced from a global pool, reducing GC pressure on frequent calls with similar-sized lists.

Implementation

void radixSortUnsigned(List<int> data, {bool reuseBuffer = true}) {
  if (data.length < 2) {
    return;
  }

  // Create a Uint32List view/copy to work with.
  final list = Uint32List.fromList(data);

  // Call the core sorting logic.
  radixSortCore(list, reuseBuffer: reuseBuffer);

  // Copy the sorted result back to the original list.
  for (var i = 0; i < data.length; i++) {
    data[i] = list[i];
  }
}
